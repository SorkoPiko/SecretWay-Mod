# Secret Way Mod
A "simple" Geometry Dash mod that adds badges and info pages to levels with secret ways.
It uses my own endpoint, hosted [here](http://secretway.sorkopiko.com) ([Source](https://github.com/SorkoPiko/GDSW-API)).
